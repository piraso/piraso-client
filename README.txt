Piraso Client

This is the Piraso client created using Netbeans Platform.

https://github.com/alvinrdeleon/piraso/wiki



Quick Start to build and run:

1. Install Maven 2 and JDK 6

2. To build source code go to ${piraso-client.home} and execute the following command:
    > mvn clean install

3. To run go to ${piraso-client.home}/application
    > mvn nbm:run-platform
